##### [Dev Day Holiday Edition—12 Days of OpenAI: Day 9](https://www.youtube.com/watch?v=14leJ1fg4Pw) #####

 * <span style="color: yellow; font-size: 10">1. 새로운 o1 정식 출시</span>
     1) o1 모델의 API 정식 출시. 이 모델은 에이전트 애플리케이션을 지원하며, 고객 지원, 재무 분석 등 다양한 분야에서 활용될 수 있음. 특히, 코딩 모델에서 뛰어나며, 이전 버전인 o1 Preview에 비해 60% 적은 thinking tokens을 사용해 더 빠르고 저렴함.
     2) 함수 호출(Function Calling): o1 함수 호출 API 제공. o1 모델이 백엔드 API와 상호 작용할 수 있으며, 개발자들은 모델이 외부 데이터를 활용하거나 특정 작업을 수행할 수 있음. 특히, o1 모델은 함수 호출에서 GPT-4o보다 성능이 훨씬 뛰어남.
     3) 구조적 출력 (Structured Outputs): 모델의 출력을 JSON 스키마에 맞춰 구조화하는 기능이 추가했음. 개발자들은 모델로부터 원하는 형식의 데이터를 쉽게 추출할 수 있으며, 특히, 풍부한 기능을 가진 클라이언트 응용 프로그램을 개발하는 데 유용함. o1 모델은 구조적 출력에서도 GPT-4o보다 성능이 뛰어남.
     4) 새 개발자 메시지 도입: 시스템 메시지의 새로운 형태인 개발자 메시지가 도입으로 개발자 메시지를 통해 모델이 어떤 지시를 어떤 순서로 따라야 하는지 제어할 수 있으며, 모델의 동작을 개발자가 완전히 제어할 수 있음.
     5) 추론 노력 (Reasoning Effort): 모델이 문제 해결에 얼마나 시간을 할애할지 제어하는 새로운 매개 변수가 도입되었습니다 [14, 15]. 이를 통해 쉬운 문제에는 시간과 비용을 절약하고, 어려운 문제에는 더 많은 컴퓨팅 자원을 사용할 수 있습니다 [15].
